== Description ==

fwd slider let you add no of images to the slider. You can add number of sliders and can use multiple slider in single page using do_shortcode.
its perfect, dynamic because you can control height ,width and even rotation speed of your sliders. So configuration is upto your need. This Dynamic slider give you your slider preview at th back-end quickly after creating them. its very simple to use ,  


		
= Features of this plugin =

*   Easy to customize.
*   Support all browser.
*   Short code available for pages and posts.
*   Hyperlink option to each images.
*   Image order option available.
*   option to set rotation speed
*   Easy to edit and delete your created slider
*   Preview for each created slider
*   Dynamicaly control its height ,with and rotation speed

We have three option to install and configure this Image horizontal reel scroll slideshow plugin.
